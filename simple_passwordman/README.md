# 🔐 Simple Password Manager (Bash)

This is a lightweight command-line password manager built in Bash.  
It allows you to store, encrypt, and retrieve passwords for different services.

---

## 📦 Features

- Store passwords for services like Gmail, Facebook, etc.
- Base64 encoding for simple obfuscation (not secure encryption)
- View stored passwords (decoded)
- File-based storage in `password.txt`

---

## 🚀 Usage

```bash
chmod +x simple_passwordman
./simple_passwordman
